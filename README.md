# Expo-notification

## To exec the server

Requirements

- Nodejs
- NPM
- Git

First clone the repository and change the directory

```
$ git clone https://github.com/andreaagt/Expo-notification.git
$ cd Expo-notification
```

Save and install all packages

```
$ npm install
```

Then Exec

```
$ npm run dev
```

This code is using modern javascript (ES6)
